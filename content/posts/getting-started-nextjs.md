# Next.js 入门指南

Next.js 是一个用于生产环境的 React 框架，它让构建全栈 Web 应用变得简单而高效。

## 为什么选择 Next.js？

Next.js 提供了许多开箱即用的功能：

1. **服务器端渲染 (SSR)**
   - 更好的 SEO
   - 更快的首屏加载

2. **静态站点生成 (SSG)**
   - 极致的性能
   - 降低服务器负载

3. **文件系统路由**
   - 基于文件的直观路由系统
   - 支持动态路由
